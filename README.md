# Iris Flower Classification

A machine learning classification project developed using the Iris dataset to identify and classify flower species. The project utilizes Decision Tree and K-Nearest Neighbors (KNN) algorithms for model training, evaluation, and species prediction.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Features

* Data preprocessing and analysis
* Exploratory data visualization
* Decision Tree classification
* K-Nearest Neighbors (KNN) classification
* Model training and evaluation
* Multi-class flower species prediction

## Dataset

The project uses the Iris dataset, which contains measurements of flower characteristics used to classify three different Iris species.

## Results

Multiple classification algorithms were implemented and evaluated to assess their effectiveness in accurately identifying flower species.
